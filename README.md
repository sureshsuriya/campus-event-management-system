# 🎓 Campus Event Management & Hall Booking System

A full-stack web application designed to simplify campus event management and hall booking processes with a modern and user-friendly interface.

---

## 🚀 Overview

This system allows students and staff to manage events, book halls, and track activities efficiently. It provides a seamless experience with role-based access and real-time data handling.

---

## 🛠️ Tech Stack

### Frontend

* ⚛️ React (Vite)
* 🎨 CSS

### Backend

* ☕ Spring Boot
* 🔐 REST APIs

### Database

* 🍃 MongoDB

---

## ✨ Features

* 🔐 User Authentication (Login / Register)
* 👨‍🎓 Student Dashboard
* 👨‍💼 Staff Dashboard
* 📅 Event Creation & Management
* 🏫 Hall Booking System
* 🔄 Real-time Data Handling
* 🌐 REST API Integration

## ⚙️ Installation & Setup

### 🔹 Clone Repository

```bash
git clone https://github.com/sureshsuriya/campus-event-management-system.git
cd campus-event-management-system
```

---

### 🔹 Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

### 🔹 Backend Setup

```bash
cd backend
mvn spring-boot:run
```
